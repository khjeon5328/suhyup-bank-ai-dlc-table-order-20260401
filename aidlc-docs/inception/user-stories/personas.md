# 테이블오더 서비스 - 사용자 페르소나

---

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| 이름 | 김민수 (대표 페르소나) |
| 역할 | 매장 방문 고객 |
| 연령대 | 20~60대 (폭넓은 연령층) |
| 기술 수준 | 다양함 (스마트폰 기본 사용 가능 수준) |
| 사용 기기 | 매장 테이블에 설치된 태블릿 |
| 사용 빈도 | 매장 방문 시마다 (비정기적) |

**목표**:
- 대기 없이 빠르게 메뉴를 주문하고 싶다
- 메뉴 정보(사진, 가격, 설명)를 충분히 확인하고 선택하고 싶다
- 현재 주문 상태를 실시간으로 알고 싶다

**불만 요소**:
- 복잡한 UI, 느린 반응 속도
- 주문이 제대로 들어갔는지 확인이 안 될 때
- 직원을 불러야 하는 상황

**핵심 니즈**:
- 직관적이고 터치 친화적인 인터페이스
- 빠른 주문 프로세스 (최소 탭 수)
- 주문 상태 실시간 확인

---

## Persona 2: 점주 (Owner)

| 항목 | 내용 |
|------|------|
| 이름 | 박영희 (대표 페르소나) |
| 역할 | 매장 소유자/최고 관리자 |
| 연령대 | 30~50대 |
| 기술 수준 | 중간 (기본 웹 서비스 사용 가능) |
| 사용 기기 | PC 또는 태블릿 (관리자 화면) |
| 사용 빈도 | 매일 (영업 시간 중 수시) |

**목표**:
- 매장 운영을 효율적으로 관리하고 싶다
- 메뉴를 자유롭게 등록/수정/삭제하고 싶다
- 직원(매니저) 계정을 관리하고 싶다
- 주문 현황을 실시간으로 파악하고 싶다

**불만 요소**:
- 복잡한 관리 도구
- 실시간 주문 현황 파악이 어려울 때
- 직원 권한 관리가 안 될 때

**권한 범위** (전체 권한):
- 메뉴 관리 (CRUD)
- 주문 모니터링 및 상태 변경
- 주문 삭제 (직권 수정)
- 테이블 관리 (초기 설정, 세션 처리)
- 관리자 계정 생성/관리
- 과거 주문 내역 조회

---

## Persona 3: 매니저 (Manager)

| 항목 | 내용 |
|------|------|
| 이름 | 이준호 (대표 페르소나) |
| 역할 | 매장 운영 매니저/직원 |
| 연령대 | 20~40대 |
| 기술 수준 | 중간~높음 |
| 사용 기기 | PC 또는 태블릿 (관리자 화면) |
| 사용 빈도 | 매일 (영업 시간 중 상시) |

**목표**:
- 들어오는 주문을 빠르게 확인하고 처리하고 싶다
- 테이블 상태를 효율적으로 관리하고 싶다
- 고객 이용 완료 후 테이블을 빠르게 리셋하고 싶다

**불만 요소**:
- 주문 알림이 늦을 때
- 테이블 상태 파악이 어려울 때

**권한 범위** (제한된 권한):
- 주문 모니터링 및 상태 변경 ✅
- 테이블 세션 처리 (이용 완료) ✅
- 과거 주문 내역 조회 ✅
- 메뉴 관리 ❌ (점주만 가능)
- 주문 삭제 ❌ (점주만 가능)
- 관리자 계정 관리 ❌ (점주만 가능)

---

## 페르소나 간 관계

```
+--------------------------------------------------+
|                   시스템                           |
|                                                    |
|  [고객] ---주문---> [백엔드] <---모니터링--- [점주] |
|                       ^                            |
|                       |                            |
|                   모니터링                          |
|                       |                            |
|                    [매니저]                         |
|                                                    |
|  점주: 전체 권한 (메뉴/주문/테이블/계정 관리)      |
|  매니저: 제한 권한 (주문 모니터링/테이블 세션 처리) |
|  고객: 주문 권한 (메뉴 조회/장바구니/주문/내역)    |
+--------------------------------------------------+
```
