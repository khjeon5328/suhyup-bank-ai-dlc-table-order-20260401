## BR-FA01: 역할 기반 UI
- 점주: 모든 메뉴 표시 (대시보드, 메뉴관리, 테이블관리, 계정관리)

## BR-FA02: 세션 관리
- JWT 16시간 세션, localStorage 저장
- 만료 시 자동 로그아웃 → 로그인 화면

## BR-FA03: 주문 상태 전이
- 대기중→준비중→완료 (순방향만)
- 주문 삭제는 점주만 가능

## BR-FA04: SSE 실시간
- 신규 주문: 테이블 카드에 즉시 반영 + 시각적 강조
- 상태 변경: 즉시 반영
- 주문 삭제: 즉시 반영
